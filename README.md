# Simulacion-de-distintos-GPs

Este proyecto se basa en la simulación y la visualización de los procesos gaussianos (GP). Estos son un tipo de proceso estocástico, $\{X_t\}_{t\in T}$, que tiene una distribución normal multivariante sobre cualquier subconjunto finito de $T$. Concretamente, en el archivo se puede ver la implementación de funciones empleadas para generar trayectorias asociadas a distintos tipos de procesos Gaussianos (GP), facilitando así la visualización de múltiples trayectorias asociadas a estos. En particular, trataremos con los procesos Gaussianos con kernel RBF, kernel periódico y kernel DPK.

### Contenido del proyecto:

  * Simulación: Generación de trayectorias aleatorias.
  * Visualización: Gráficas asociadas a los distintos procesos utilizando matplotlib.

### Recursos utilizados:

 * Lenguaje: Jupyter Notebook.
 * Librerías:
    * Numpy.
    * Matplotlib.
    * Math
